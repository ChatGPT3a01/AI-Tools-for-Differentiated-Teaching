📋 提示詞總整理：一句話取得完整工具

以下是三個工具的「一句話提示詞」，貼到任何 AI 都能用：

工具 ① 段考 AI 審題助手

請幫我寫一個 Google Apps Script，讀取 Google Sheets A 欄的段考題目，呼叫 Gemini API 分析，自動產出雙向細目表（單元×認知層次）、難易度分析（簡單/中等/困難統計）、檢核與建議，分別寫入三個工作表。要有自訂選單、JSON 格式回傳、temperature 0.2。

工具 ② 動態評量產生器

請幫我寫一個 Google Apps Script，讀取 Sheets 的科目/單元/知識點，呼叫 Gemini API 為每個知識點產生一道應用題和三層漸進式提示（方向→公式→帶數字），然後自動建立含分支邏輯的 Google Form，計分邏輯：0提示=5分、1提示=4分、2提示=3分、3提示=1分。

工具 ③ AI 學生回饋系統

請幫我寫一個 Google Apps Script，讀取 Sheets 的學生姓名、評量形式、Rubric、作品內容，呼叫 Gemini API 產生個人化回饋（逐項評分、優點、缺點、改進建議、鼓勵語）。Prompt 要強制 AI 誠實指出缺點，不能只說好話。支援單筆和批次產生。
🎯 重點：這些提示詞是「起點」，AI 產出後你可以繼續對話調整。
例如：「請把 Prompt 改成適合國文科的版本」「請加入匯出 PDF 的功能」

🔒 API 金鋀安全須知

⚠️

勿分享含金鋀的檔案

不要把 .gs 檔案直接傳給別人！

金鋀如果寫死在程式碼裡，分享檔案等於分享金鋀。

✅

用「指令碼屬性」存放

Apps Script 內建安全儲存：

PropertiesService
.getScriptProperties()

金鋀不會出現在程式碼中。

✅

定期檢查 API 用量

Google AI Studio → API Keys
→ 查看用量與額度

確認是否有異常使用。

💡 如果金鋀外洩怎麼辦？
不用緊張！到 Google AI Studio 重新產生一組新金鋀，舊的就會自動失效。
🔐 記住一句話：金鋀就像家裡的鍵匙，不要隨便給別人。

📝 Prompt 設計速查表

情境 Prompt 技巧 範例
要 AI 分析結構化資料 指定 JSON 輸出格式 「請用以下 JSON 格式回傳...」
要 AI 扮演特定角色 角色設定 + 專業約束 「你是一位嚴謹的高中物理老師」
要 AI 誠實批評 明確要求找缺點 「請先指出不足，如果要扣分在哪裡扣？」
要 AI 依照標準評分 提供 Rubric + 逐項要求 「根據以下 Rubric 逐項評估」
要 AI 簡化內容 指定目標讀者 + 字數限制 「改寫為高一生能懂的版本，每句不超過 20 字」
要 AI 產出多層次內容 明確描述層次差異 「提示1只給方向，提示2給公式，提示3帶數字」
降低 AI 的隨機性 設定低 temperature temperature: 0.2（分析用）
增加 AI 的創意 設定高 temperature temperature: 0.7（改寫用）

❓ 常見問題 FAQ

+ Q1：API 金鋀要付費嗎？
Gemini 2.5 Flash 有免費額度，一般教學使用絼絼有餘。超過也很便宜（約每百萬字元 $0.15 美元）。
+ Q2：學生資料丢給 AI 安全嗎？
Google 官方聲明 API 傳送的資料不會用於訓練模型。建議：不要傳身分証號等敏感個資，姓名可用座號代替。
+ Q3：AI 產出的雙向細目表準確嗎？
大方向是準確的，但建議老師仍需人工審核。AI 的偏差正好可以作為反思出題的切入點。
+ Q4：動態評量可以用在哪些科目？
有計算或明確解題步驟的科目最適合（數學、物理、化學）。文科也可以用，只是提示的設計方式不同。
+ Q5：我不會寫程式，可以用嗎？
可以！只需要：會用 Google Sheets、會開 Apps Script（就是點一個選單）、會複製貼上、會申請 API 金鋀（有步驟教你）。
