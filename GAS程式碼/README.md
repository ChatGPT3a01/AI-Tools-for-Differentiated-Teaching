# GAS 程式碼（Google Apps Script）

這個資料夾放的是研習用到的三個工具程式碼。

**你不需要看懂程式碼！** 只要會「複製 → 貼上」就能用。

---

## 資料夾裡有什麼？

| 檔案 | 對應工具 | 做什麼用 |
|------|---------|---------|
| `工具1_段考AI審題助手.js` | 工具 ① | 貼上段考題 → AI 產出雙向細目表 + 難易度分析 |
| `工具2_動態評量產生器.js` | 工具 ② | 輸入知識點 → AI 產出題目 + 三層提示 → 自動建 Google Form |
| `工具3_AI學生回饋系統.js` | 工具 ③ | 貼上學生作品 + Rubric → AI 產出個人化回饋 |

---

## 怎麼用？（三個工具都一樣的步驟）

```
步驟 1 → 新增一份 Google Sheets
步驟 2 → 點「擴充功能」→「Apps Script」
步驟 3 → 把對應的 .js 檔案內容「全選 → 複製 → 貼上」
步驟 4 → 設定 API 金鑰（齒輪 → 指令碼屬性 → GEMINI_API_KEY）
步驟 5 → 儲存 → 執行 onOpen → 授權
步驟 6 → 回到 Sheets 按 F5 → 上方出現新選單 → 開始用！
```

---

## API 金鑰哪裡拿？

1. 前往 https://aistudio.google.com/apikey
2. 點「建立 API 金鑰」
3. 複製金鑰（一串 AIza... 開頭的文字）
4. 貼到 Apps Script 的「指令碼屬性」裡

> 金鑰 = 密碼，不要分享給別人、不要貼在公開文件裡！
